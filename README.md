# Expense Tracker (Трекер расходов)

**Автор:** [Галиева Адиля]

## Описание
GUI-приложение на Python/Tkinter для отслеживания расходов. Поддерживает добавление записей, фильтрацию по категории/дате, подсчёт суммы за период, сохранение в JSON.

## Установка и запуск
1. Установите Python 3.8+.
2. `pip install tkinter` (обычно встроено).
3. Запустите: `python proekt_4.5.py`.

## Как использовать
- Введите сумму (положительное число), категорию, дату (YYYY-MM-DD).
- Нажмите "Добавить расход".
- Используйте фильтры и "Сумма за период".

## Примеры
- Добавьте: 500, "еда", 2026-05-04. Сумма за день: 500.
- Фильтр "еда": показывает только еду.
